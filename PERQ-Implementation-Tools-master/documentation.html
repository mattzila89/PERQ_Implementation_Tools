View Changer

Enables different view options, since all of the tools in the extension are only used by certain teams. It will remember your view even when closed/reopened.

Highlight CSSD Hair on Fire

Sets a light red background on tickets with a Hair on Fire priority throughout the CSSD Jira board.

Show CI Jira Colors

Sets varying background colors on tickets in the CI Jira board depending on due date as shown below.

  • overdue
  • due today
  • due this week
  • due in two weeks
  • due in three weeks
  • due in four weeks
  • further out than four weeks

Drip Account Search

Adds a search field to the top of the account list in Drip. The field will compare the search query against any part of a given account and display it in the results if the account matches.

You can Tab through results and select an account with Enter.

This feature also forces the navigation bar to be sticky, so it is not hidden when scrolled further down the page.

Blink CSSD Hub and Lead Nurture

The "Multifamily Hub" and "Client Has Lead Nurture" fields will blink when viewing a CSSD ticket if the ticket has either of those options marked. This helps ensure these areas are not missed when working the ticket.

Milkyway Keyboard Shortcuts

Enables Ctrl+Shift+S keyboard shortcut to open the client search window. Note you can also Tab through results and use Enter to select them, but this is a native feature of Milkyway.

Email Log Enhancements

Improves the Email Log tool to show visited links and highlight rows on hover.

Check for Fatwin Code

Automatically checks for Fatwin code on every page. If code is found, a green checkmark will appear on this extension's icon.

Note: some web providers may have issues with this feature, so if you're expecting to see code up be sure to double check using the standard method.

Check for Iframe on Page

This will allow you to check if floorplan pages contain an iframe for proper integration, or not. If an iframe is found on the page, a red border will appear around the iframe, and an alert message will pop-up.

Note: Some floorplan pages may still contain iframes, but they may be outside of the floorplan section. So be sure to double check if you see the warning but no red border popping up around the floorplan section. If you are unsure, you can also check by right clicking on the floorplan section and there will be an option to reload the iframe (signifying that the section is in fact within an iframe)

Virtual Tour Formatter

Enables the plugin team to quickly format URLs and associated values for virtual tour builds. Once URLs and values have been input into their associated fields, the formatted result can be simply copied and then cleared for more builds.

Lead Routing Formatter

Enables the plugin team to quickly format the field input values from the promotion, and the emails associated with each input field value. Like the Virtual Tour formatter, the formatted result can be simply copied and then cleared for more builds.

DRIP URL Formatter

Enables the plugin team to quickly format the URLs for the Jira table. Like the other formatters, the formatted result can be simply copied for use in Jira.

Note: Only for Furniture clients at the moment. Be sure URLs are put in this same order every time -

  • Client Name
  • Design Assessment
  • Design Consultation
  • Chair Consultation
  • Mattress Consultation
  • Sofa Consultation

Resizing Plugin Window

Enables you to have the option to automatically resize the custom CSS/JS fields when navigating to the details page of a plugin for better viewing and editing.

Note: Page will automatically refresh for you to implement the changes upon checking the option in the extension.

Show PTSD Jira Colors

Sets varying background colors on tickets in the PTSD Jira board depending on time to resolution, as shown below.

  • Overdue
  • Due in less than 1 hour
  • Due in 1+ hour
  • Due in 2+ hours
  • Due in 3+ hours
  • Due in 4+ hours

HUB Experience Testing

When feature is turned on in the extension, it will allow you to test different web experiences within the Hub. When one of the experience buttons is clicked on, the the experience will pop-up from the Hub.

    Experiences included:

    Exit Intent

    Expense Calculator

    Floor Plan Match

    New Customer Welcome

    Special Offers

    Reset Local Storage

    Selector Tool

Along with the hub experiences, you can also click and drag the settings tool anywhere you'd like on the page for better access.

Note: Fatwin extension must be turned on, if client isn't live. If you switch to mobile view and you can't see the tool in the bottom corner, then you'll just need to refresh and it'll show up.

Selector Tool

Allows you to select various elements on the page and finding out their selectors. You can click the copy button to copy the selector for easy accessibility, and then clear or exit out right next to the selector field.

It defaults to showing shared selectors, but if you deselect the shared elements until it's only showing one selector, then you'll have the unique selector you're looking for.
